Ответьте на вопросы:

****************************************************
Что такое Apache Hadoop?

"Это платформа, которая позволяет распределять обработку больших наборов данных между кластерами компьютеров с использованием простых моделей программирования. Является проектом с открытым исходным кодом, предназначенный для эффективной обработки больших пакетов данных."

****************************************************
Что такое HDFS?

"распределенная файловая система Hadoop"

****************************************************
Что такое Apache Hive?

"Это система управления базами данных на основе платформы Hadoop, предназначена для обработки данных (запросы, агрегирование и анализ), хранящиеся в Hadoop."

****************************************************
Какие задачи можно решить с помощью Spark?

"Предназначен для кластерных вычислений, параллельной обработки неструктурированных или слабоструктурированных данных,Решает задачи: изучение потребительского поведения, прогнозирование прибыли и финансовых рисков, обработка данных сенсоров и датчиков в системе интернет вещей, анализ информации о транзакциях, безопасности финансовых операций и утечек."
Из чего состоит и как работает Spark?

"Состоит из: 1) ядро (движок) - отвечает за хранение данных, управление памятью, распределение и отслеживание задач в кластере; 2) MLlib - набор библиотек для машинного обучения; 3) GraphX - модуль для работы с графами; 4) Streaming - средство для обработки потоков Big Data в реальном времени. Spark интегрирован в Hadoop"
